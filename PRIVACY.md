Vita Graf Eisen - Privacy Policy
1. Data Collection
Vita Graf Eisen collects minimal data necessary for auditing and system stability:

Audit Logs: Command names, user tags, and server names are recorded in the vitabot-logs channel to monitor system health.

Error Handling: Technical errors are logged to facilitate debugging on the host Windows Server 2025.

2. Data Security & Sanitization

Path Protection: Our logging system automatically sanitizes error messages, removing local directory paths (C:\Users...) to protect the privacy and structure of the host server.

No Third-Party Sharing: No data is sold or shared with third parties. Gemini AI interactions are processed via Google's API securely.

3. Storage
All audit logs are stored locally on a private server located in Toluca/Metepec, Mexico.

4. Data Deletion
If the Bot is removed from a server, the audit channel and associated local cache entries can be deleted by the administrator using the /config command or manual deletion.
